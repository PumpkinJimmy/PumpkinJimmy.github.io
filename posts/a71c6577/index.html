¶ 单位脉冲函数

连续情况较不直观：

规定\delta(t)满足如下性质：\\ \delta(t) = 0, t \ne 0\\ \int_{-\infty}^{\infty} \delta(t) dt = 0

δ(t)\delta(t)δ(t)是奇异函数，实际上没有办法说清楚δ(0)\delta(0)δ(0)的是多少

筛选性质：

¶ 卷积

卷积运算：

f[n] * g[n] = \sum_{k=-\infty}^{\infty} f[k] g[n-k]\\ f(t) * g(t) = \int_{-\infty}^{\infty} f(\tau)g(t-\tau)\mathrm{d}\tau

卷积运算的性质：

  • 交换律

  • 结合律（从交换律推出）

  • 分配律

  • 微分性质

    (x(t)∗y(t))′=x′(t)∗y(t)=x(t)∗y′(t) (x(t)*y(t))' = x'(t) * y(t) = x(t) * y'(t) (x(t)∗y(t))​′​​=x​′​​(t)∗y(t)=x(t)∗y​′​​(t)

    ∫(x(t)∗y(t))=∫(x(t))∗y(t)=x(t)∗∫y(t) \int (x(t)*y(t)) = \int (x(t)) * y(t) = x(t)*\int y(t) ∫(x(t)∗y(t))=∫(x(t))∗y(t)=x(t)∗∫y(t)

  • 时不变

    若y(t) = x(t) * h(t)\\ 则y(t-t_0) = x(t-t_0)*h(t) = x(t)*h(t-t_0)

  • 对离散情况同样有差分、求和和时不变

¶ LTI System

LTI(Linear Time Invarient) System 线性时不变系统

单位脉冲响应是指系统对单位脉冲信号输入的相应

信号的线性分解：

任何信号都可以表示成时移单位脉冲信号的线性组合，即：

x[n]=∑k=−∞∞x[k]δ[n−k] x[n] = \sum_{k=-\infty}^{\infty} x[k] \delta[n-k] x[n]=​k=−∞​∑​∞​​x[k]δ[n−k]

实际上，由卷积公式可知，上式等价于：

x[n]=x[n]∗δ[n] x[n] = x[n] * \delta[n] x[n]=x[n]∗δ[n]

结合信号的线性分解、线性时不变系统的性质，有：

线性时不变系统的响应公式：

设h[n]表示系统单位脉冲响应\\ 系统对输入x[n]的响应y[n]有：\\ y[n] = x[n] * h[n]\\ 同理连续情形下有：\\ y(t) = x(t) * h(t)

推论：一个线性时不变系统可以由其单位脉冲响应完全表征。因为这一个推论，又是会直接写出单位脉冲响应来代表一个LTI系统

写成泛函：

J(x(t))=x(t)∗J(δ(t)) \mathcal{J}(x(t)) = x(t) * \mathcal{J}(\delta(t)) J(x(t))=x(t)∗J(δ(t))

线性时不变系统互联的性质（以h1(t),h2(t)h_1(t), h_2(t)h​1​​(t),h​2​​(t)为例）：

  • 并联：h1(t)+h2(t)h_1(t) + h_2(t)h​1​​(t)+h​2​​(t)
  • 级联（串联）：h1(t)∗h2(t)h_1(t) * h_2(t)h​1​​(t)∗h​2​​(t)

注意上述结论可以由卷积运算的性质得出

从单位脉冲响应研究LTI的系统性质

  • 记忆性：

    当h(t)=kδ(t)h(t) = k\delta(t)h(t)=kδ(t)时LTI无记忆，否则是有记忆的

  • 可逆性：

    LTI系统若可逆，则其逆系统也是LTI

    由LTI级联性质可知若h(t),g(t)h(t),g(t)h(t),g(t)系统互逆，则必有：

    h(t)∗g(t)=δ(t) h(t) * g(t) = \delta(t) h(t)∗g(t)=δ(t)

  • 因果性：

    若h(t)h(t)h(t)满足h(t)=0,t<0h(t)=0, t < 0h(t)=0,t<0，则该LTI是因果的，否则是非因果的

    几何解释：把h(t)h(t)h(t)镜像反转，然后在x(t)x(t)x(t)上平移，发现当t≥0t\ge 0t≥0时h(t)h(t)h(t)的值就是对当前点过去和现在的加权累积的权值，相应的t<0t<0t<0时h(t)h(t)h(t)时“未来的权值”。未来的权值为0即与未来取值无关，即“因果的”

¶ 常微分方程描述的LTI系统

此处研究的常微分方程是描述了输出函数y(t)y(t)y(t)及其导数与输入函数x(t)x(t)x(t)及其导数的方程。

这样的微分方程实际上表征了一个从从x(t)x(t)x(t)到y(t)y(t)y(t)的函数算子。如果认为x(t),y(t)x(t),y(t)x(t),y(t)分别是输入、输出信号，则微分方程表征了一个系统。

但是，类似常微分方程初值问题，微分方程配合一个初始条件才能唯一确定一个系统。

考虑线性常系数微分方程，有如下结论：

具有初始松弛条件的线性常系数微分方程确定了一个LTI系统。

（离散情形下差分方程有类似结论）

¶ 初始松弛条件

初始松弛条件：系统具有“0输入-0输出”的响应特点

符号表示：

∀t<t0(x(t)=0)→∀t<t0(y(t)=0) \forall t < t_0(x(t) = 0) \to \forall t < t_0(y(t) = 0) ∀t<t​0​​(x(t)=0)→∀t<t​0​​(y(t)=0)

对于LTI系统，一种等价表示是：h(t)h(t)h(t)是因果信号

初始松弛条件必要性的一种直观情景是考虑系统y(t)=x(t)+2y(t) = x(t) + 2y(t)=x(t)+2，它具有初始的增量，导致系统满足可乘性，不满足齐次性。

初始松弛条件的一种物理对应是零初始状态系统，例如含容电路，这里的初始松弛条件等价于要求具有记忆性的电路元件初始都是“0状态”（电容初始没有电荷）。

使用傅里叶级数的动机有两部分：

  1. LTI系统对复指数信号的响应具有简单的形式
  2. 一大类常见的周期信号可以表示为复指数信号的线性组合

    LTI对复指数信号este^{st}e​st​​的响应有如下形式：

    其中，s=a+bjs=a+bjs=a+bj为复数

¶ 傅里叶级数存在的条件

充要条件很难找，但有两组充分条件

  1. 平方可积条件

    若∫−∞+∞∣x(t)∣2dt<∞\int_{-\infty}^{+\infty}|x(t)|^2dt < \infty∫​−∞​+∞​​∣x(t)∣​2​​dt<∞（其实就是信号能量有限）

  2. 迪利克雷条件

    三条：

    1. 绝对可积
    2. 任意有限区间，x(t)x(t)x(t)只有有限个极值点，且极值有限
    3. 任意有限区间，x(t)x(t)x(t)只有有限个间断点、且间断点上函数值有限

可以发现，某些具有有限个间断点的函数依然有傅里叶级数存在（比如周期方波），且傅里叶级数在间断点处依然有定义，但实际上在间断点附近的傅里叶级数的取值是怪异的（可能会反复震荡）。这种现象称为吉布斯现象

¶ 频率响应 & 信号失真

LTI频率响应H(jω)H(j\omega)H(jω)是一个复变函数，其特征难以感知和可视化。常对其做分解得到两组实函数：

  1. ∣H(jω)∣|H(j\omega)|∣H(jω)∣称为幅频特性
  2. ∠H(jω)\angle H(j\omega)∠H(jω)称为相频特性

实际上，无论对信号的幅度还是相位引入变化，都有可能导致信号波形的显著变化，我们称为信号失真

由LTI系统响应公式Y(jω)=H(jω)X(jω)Y(j\omega) = H(j\omega)X(j\omega)Y(jω)=H(jω)X(jω)有如下事实：

|Y(j\omega)| = |H(j\omega)||X(j\omega)|\\ \angle Y(j\omega) = \angle H(j\omega) + \angle X(j\omega)

如果要保证一个LTI系统不会导致系统失真，显然要求频率响应的幅度为非零常数，相位为常数。这样可以保证系统对输入的变换为：

  1. 对输入信号的幅度做等幅缩放
  2. 对输入信号做时域平移

以上两点变化都不会修改输入波的形状。因此LTI系统不失真条件为：

H(jω)=ke−jt0ω H(j\omega) = ke^{-jt_0\omega} H(jω)=ke​−jt​0​​ω​​
