¶ Ordinary Differential Equation

¶ Definition

常微分方程是指关于一个未知函数及其任意阶全导数的方程。

所谓常微分方程区别于偏微分方程，不涉及偏导数

常微分方程的阶数表示的是方程中出现的最高阶的导数。

常微分方程有两种常见形式：

  1. 关于自变量、未知函数、以及未知函数的方程形式

    通式，基本上就是把定义复述了一遍：

    f(y(n),y(n−1),⋯,y′,y,x)=0f(y^{(n)}, y^{(n-1)}, \cdots, y', y, x) = 0f(y​(n)​​,y​(n−1)​​,⋯,y​′​​,y,x)=0

  2. 关于输入函数和输出函数及其导数的方程（常见于系统分析相关的理论）

    这种形式实际上是将一个关于输入函数和输出函数的算子表征为一个微分方程的形式。

    此处不涉及这种形式

¶ 一阶常微分方程

一阶常微分方程理论是相对最完善的。

许多高阶常微分方程都可以变换为一阶常微分组。

ODE数值方法最基础的就是解一阶常系数微分方程y′=f(t,y)y' = f(t,y)y​′​​=f(t,y)

（也即数值方法不能直接应用在解一般的一阶常系数微分方程F(t,y,y′)=0F(t,y,y') = 0F(t,y,y​′​​)=0）

¶ ODE Method Critiaria

全局末端误差（Final Global Error, F.G.E）是指微分方程数值解在末端点取值与理论解的误差。

¶ Euler’s Method

用于解特定形式的一阶常微分方程（包括线性和非线性的）：

y′=f(y,t) y' = f(y,t) y​′​​=f(y,t)

（也即不能用于F(y′,y,t)=0F(y',y,t)=0F(y​′​​,y,t)=0形式的微分方程）

欧拉法不是迭代法。因为**微分方程的解是一个函数，

¶ Error Analysis

Local Error of Euler Method is O(h2)O(h^2)O(h​2​​) (from Talyor Expansion)

F.G.E is O(h)O(h)O(h)

区别Local Error和F.G.E：命名都是数值解和解析解之间的误差，为什么末端点数值解与解析解的Local Error是O(h2)O(h^2)O(h​2​​)，但末端点数值解与解析解之间的误差F.G.E却是O(h)O(h)O(h)？

这是因为：

  • Local Error是若前一点是准确值，计算一部的误差
  • 而F.G.E是准确的初值，经过若干步计算，末端的误差
  • 每一步都是O(h2)O(h^2)O(h​2​​)都是

¶ Method

Henu Method和Euler Method的共同框架：

yk+1=yk+ϕh y_{k+1} = y_k + \phi h y​k+1​​=y​k​​+ϕh

欧拉法其实就是令ϕ=f(tk,yk)\phi = f(t_k, y_k)ϕ=f(t​k​​,y​k​​)

Henu Method的改进是改进ϕ\phiϕ的选择，采用迭代法选择更好的ϕ\phiϕ

初始ϕ\phiϕ同欧拉法：

yk+10=yk+f(tk,yk)h y_{k+1}^0 = y_k + f(t_k,y_k)h y​k+1​0​​=y​k​​+f(t​k​​,y​k​​)h

然后取平均，使用新的ϕ\phiϕ得到一步迭代结果：

yk+11=yk+f(tk,yk)+f(tk+1,yk+10)2h y_{k+1}^1 = y_k + \frac{f(t_k, y_k) + f(t_{k+1}, y_{k+1}^0)}{2} h y​k+1​1​​=y​k​​+​2​​f(t​k​​,y​k​​)+f(t​k+1​​,y​k+1​0​​)​​h

反复迭代nnn次到收敛，得到一个数值结果：

yk+1=yk+1n y_{k+1} = y_{k+1}^n y​k+1​​=y​k+1​n​​

接下类，类似欧拉法，逐步前进即可。

¶ Runge-Kutta Methods (RK Methods)

Runge-Kutta Methods是一个框架，可以从这个框架导出Euler Method, Heun Method, 还至少可以导出O(h2)O(h^2)O(h​2​​)到O(h6)O(h^6)O(h​6​​)的算法.

算法框架核心仍然是：

RK Method 给出了系统的计算ϕ\phiϕ的方法。

¶ System of ODE

很多时候，可以将高阶微分方程转写为一阶微分方程组。

通常，若nnn阶常微分方程组有如下通式：

y(n)=f(t,y,y′,y′′,⋯,y(n−1)) y^{(n)} = f(t,y,y',y'', \cdots, y^{(n-1)}) y​(n)​​=f(t,y,y​′​​,y​′′​​,⋯,y​(n−1)​​)

{y1=yy2=y′y3=y′′⋮yn=y(n−1) \begin{cases} y_1 = y\\ y_2 = y'\\ y_3 = y''\\ \vdots\\ y_n = y^{(n-1)} \end{cases} ​⎩​⎪​⎪​⎪​⎪​⎪​⎪​⎨​⎪​⎪​⎪​⎪​⎪​⎪​⎧​​​y​1​​=y​y​2​​=y​′​​​y​3​​=y​′′​​​⋮​y​n​​=y​(n−1)​​​​

可以得到一阶常微分方程组：

{y1′=y2y2′=y3⋮yn−1′=ynyn′=f(t,y1,y2,⋯,yn) \begin{cases} y_1' = y_2\\ y_2' = y_3\\ \vdots\\ y_{n-1}' = y_n\\ y_n' = f(t,y_1,y_2,\cdots,y_n) \end{cases} ​⎩​⎪​⎪​⎪​⎪​⎪​⎪​⎨​⎪​⎪​⎪​⎪​⎪​⎪​⎧​​​y​1​′​​=y​2​​​y​2​′​​=y​3​​​⋮​y​n−1​′​​=y​n​​​y​n​′​​=f(t,y​1​​,y​2​​,⋯,y​n​​)​​

¶ 隐式的一阶常微分方程求解

考虑一半形式的一阶常微分方程：

F(t,y,y′)=0 F(t,y,y') = 0 F(t,y,y​′​​)=0

看起来无法直接使用上述的方法，因为没有y′=f(t,y)y' = f(t,y)y​′​​=f(t,y)中fff的显式表达式，无法对给定的t,yt,yt,y求值。

但实际上，对给定的t,yt,yt,y，只要将原方程中的t,yt,yt,y看作常数，y′y'y​′​​看作变量，然后原方程变为：

F(y′)=0 F(y') = 0 F(y​′​​)=0

这是关于y′y'y​′​​的非线性方程，可以直接求解方程等价于得到给定自变量的y′=f(t,y)y'=f(t,y)y​′​​=f(t,y)的函数值。（求解该方程可以使用牛顿迭代）

由此，使用牛顿法，我们可以在不求解f(t,y)f(t,y)f(t,y)的显式表达式的情况下求得其函数值，从而化归回到经典的一阶常微分求解方程。

¶ Boundary Value Problem (B.V.P)

¶ Definition & Theorm

考虑如下形式的边界值问题：

y′′=f(t,y,y′),y(a)=α,y(b)=β y'' = f(t,y,y'), y(a) = \alpha, y(b) = \beta y​′′​​=f(t,y,y​′​​),y(a)=α,y(b)=β

我们有对该问题的李普希兹判据：

f_2(t,y,y') > 0\\ |f_3(t,y,y')| < M

线性边界值问题：

x′′=p(t)x′+q(t)x+r(t),x(a)=α,x(b)=β x'' = p(t)x' + q(t)x + r(t),x(a) = \alpha, x(b) = \beta x​′′​​=p(t)x​′​​+q(t)x+r(t),x(a)=α,x(b)=β

代入李普希兹条件：

q(t) > 0\\ |p(t)| < M

我们只讨论线性边界值问题的求解：

  • Linear Shooting
  • Finite Difference

¶ Linear Shooting

利用线性边界值问题的特殊性，做如下构造：

x(t)=u(t)+Cv(t) x(t) = u(t) + Cv(t) x(t)=u(t)+Cv(t)

然后拆成两个IVP：

u = p(t)u' + q(t)u + r(t), u(a) = \alpha, u'(a) = 0\\ v = p(t)v' + q(t)v, v(a) = 0, v'(a) = 1

求解这两个IVP，然后有原问题的解：
